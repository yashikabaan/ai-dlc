# Human Roles

Last Updated: `<YYYY-MM-DD>`

## Product Owner (PO)
- Responsibilities: prioritize backlog, define value, approve acceptance.
- Decisions Owned: scope, priority, release intent.
- AI Interaction: uses Product/Planning AI for shaping and clarity.

## Designer
- Responsibilities: interaction, UX consistency, usability outcomes.
- Decisions Owned: UX direction and design tradeoffs.
- AI Interaction: uses Documentation/Workflow AI for artifacts and handoff quality.

## Frontend Engineer
- Responsibilities: client implementation, UI behavior, test coverage.
- Decisions Owned: frontend technical tradeoffs within architecture constraints.
- AI Interaction: uses Code Understanding/Implementation/Review AI.

## Backend Engineer
- Responsibilities: service logic, data handling, API behavior, reliability.
- Decisions Owned: backend implementation tradeoffs within architecture constraints.
- AI Interaction: uses Code Understanding/Implementation/Review AI.

## Senior Engineer / Lead
- Responsibilities: architecture coherence, technical risk management, mentoring.
- Decisions Owned: architectural direction, technical exceptions.
- AI Interaction: uses Review/Workflow AI for risk and process visibility.

## Scrum / Coordinator
- Responsibilities: cadence, blocker removal, cross-role coordination.
- Decisions Owned: process facilitation and escalation timing.
- AI Interaction: uses Meeting/Workflow AI for summaries and reporting.

## Shared Role Agreements
- Humans remain accountable for decisions.
- AI outputs are drafts/support material until human-reviewed.
